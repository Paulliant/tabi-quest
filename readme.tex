\section{概要}
TabiQuestは、旅行中にミッションや極秘タスクを通じて体験を拡張する、グループ向けゲーム型アプリケーションである。  
ユーザーは旅行先で様々なミッションに挑戦し、スコアを競いながら新しい体験を楽しむことができる。

\section{主な機能}
\begin{itemize}
    \item ユーザー登録・ログイン
    \item グループ作成・参加
    \item ミッションの生成・表示・達成
    \item 極秘ミッション機能
    \item スコア・ランキング表示
\end{itemize}

\section{システム構成}
本システムは以下の要素で構成される：
\begin{itemize}
    \item フロントエンド
    \item バックエンド
    \item 外部API（ChatGPT, Google Maps）
\end{itemize}

\section{使用方法}
\begin{enumerate}
    \item ユーザー登録・ログインを行う
    \item グループを作成または参加する
    \item ミッションを確認し、旅行中に実行する
    \item スコアを確認し、ランキングを競う
\end{enumerate}
