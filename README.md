# TabiQuest

TabiQuestは、旅行中にミッションや極秘タスクを通じて体験を拡張する、グループ向けのゲーム型アプリです。

---

## 🚀 概要

TabiQuestでは、旅行中に以下のような要素を楽しむことができます：

- 🎯 共通ミッション：全員で達成を目指すタスク  
- 🤫 極秘ミッション：他のメンバーに知られずに実行するタスク  
- 🧠 推理要素：他人の極秘ミッションを当てるゲーム  
- 🏆 スコアシステム：達成状況に応じたポイント付与  

---

## 🛠️ 主な機能

- ユーザー登録・ログイン
- グループ作成・参加
- ミッション生成（AI活用）
- ミッションの表示・達成管理
- 極秘ミッション機能
- スコア・ランキング表示

---

## 🧩 技術構成（予定）

- Frontend: （例）React / Next.js
- Backend: （例）Node.js / Express
- Database: （例）MySQL / PostgreSQL
- API: OpenAI API, Google Maps API

---

## 🎮 使用イメージ

1. グループを作成・参加  
2. ミッションが自動生成される  
3. 旅行中にミッションを実行  
4. 他人の極秘ミッションを推理  
5. スコアを競い、結果を楽しむ
