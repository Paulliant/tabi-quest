# TabiQuest

TabiQuestは、旅行をもっと盛り上げるためのグループ向けミッションアプリです。  
旅先で仲間とミッションに挑戦し、極秘ミッションをこっそり達成しながら、スコアとランキングを楽しめます。

## ✈️ こんな旅行に

- 友人同士の週末旅行
- ゼミ・サークルの合宿
- 家族旅行
- 卒業旅行やグループ遠征

ただ観光するだけでは終わらない、会話と発見が増える旅をつくります。

## ✅ できること

- ユーザー登録・ログイン
- グループへの参加
- 共通ミッションの確認と達成
- 極秘ミッションへの挑戦
- スコアの獲得
- フレンドランキングの確認

## 🎮 TabiQuestの楽しみ方

1. アカウントを作成してログイン
2. グループに参加して旅をスタート
3. 共通ミッションをみんなで進める
4. 極秘ミッションを気づかれないように達成する
5. ランキングを見ながら最後まで競い合う

## 🎯 ミッションの例

- 朝市で一番人気の食べ物を聞く
- 旅先らしい景色で集合写真を撮る
- 仲間を自然に褒める
- 予定にない道をひとつ提案する

旅の場所やメンバーに合わせて、行動したくなる課題が並びます。

## 🖥️ 画面イメージ

- 上部バーでアプリ名、設定、ログアウトを確認
- メイン画面で自分の表示名、現在スコア、進行状況を表示
- 左側にミッション一覧
- 右側にフレンドランキング

誰がどれだけ活躍しているかが一目でわかる構成です。

## 🌟 TabiQuestが目指す体験

TabiQuestは、旅の途中で自然に会話が生まれ、普段はしない行動に挑戦できる体験を目指しています。  
「どこへ行くか」だけでなく、「旅先で何をするか」をもっと面白くします。

## 👥 対象ユーザー

- 仲間と一緒に旅行を楽しみたい人
- 旅の思い出をもっと濃くしたい人
- ただの移動や観光では物足りない人
- ゲーム感覚で旅行を盛り上げたい人

## 🤝 チーム

TabiQuestは、グループワークから生まれたプロジェクトです。
